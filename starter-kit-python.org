* Starter kit Python

Support for the Python programming language.

** Check Dependencies

Determine whether required packages are installed. If not, use ELPA to
install them.
#+begin_src emacs-lisp
  (dolist (package '(elpy))
    (unless (package-installed-p package)
      (package-install package)))
#+end_src

#+begin_src emacs-lisp

(package-initialize)
(elpy-enable)
(elpy-use-ipython)
